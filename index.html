Prediction Image

This API will return response in JSON format, about the prediction of fruits.

Request

Method: POST

URL: /predict

Content-Type: multipart/form-data

Key&Value: uploaded_file | image.jpg/etc

Response Example

                        
                            {
                                "error" : false,
                                "message" : "Prediction successfully !",
                                "prediction_result" : {
                                    "predicted_class" : 0,
                                    "predicted_label" : "Apple",
                                    "image_url" : "https://example.com/image.jpg",
                                }
                            }
